# Coaching Methodology

## Behavior Change Principles

1. **Start small**: One habit at a time, 2-minute rule for new behaviors
2. **Stack habits**: Attach new habits to existing routines
3. **Track visibly**: Streaks and progress charts increase adherence 40%+
4. **Celebrate wins**: Acknowledge every completed session

## Motivational Interviewing Techniques

- Ask open questions: "What would success look like this week?"
- Reflect back: "It sounds like consistency is your main challenge"
- Affirm effort: "Showing up 3 days this week is real progress"
- Support autonomy: "What adjustment would work best for you?"

## Program Adaptation Triggers

- Missed 2+ sessions: Reduce volume, simplify program
- RPE consistently > 9: Reduce load 5-10%
- Pain (not soreness): Remove aggravating exercises immediately
- Plateau 3+ weeks: Change rep range or exercise variation

## Gamification Best Practices

- XP for effort (showing up), not just performance
- Streaks with forgiveness (1 miss per week doesn't break streak)
- Badges for milestones, not grind
- Social proof optional but powerful
